Tela Help File

Pekka Janhunen, Pekka.Janhunen@fmi.fi

v1.0, 2 August 1994
Tela ("Tensor Language") is a numerical computing environment. This is the basic Tela help file. It is read online by the Tela help function. You can also format and print it as a whole using the Linuxdoc-SGML 'format' command. See also: telafuncs.html, telafuncsSectioned.html. Graphics examples and FMI/GEO home page are also available. Send Tela bug reports to tela-bugs@fmi.fi. Send suggestions to tela-suggestions@fmi.fi.
